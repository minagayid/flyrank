# Decide Once: Build Your Identity Kit

## Identity

**Name mark:** Mina.Gayid  
**Monogram / favicon:** [MG monogram](assets/mg-monogram.svg)

## Type

- **Headings:** Fraunces — serif, confident, human, and suitable for a clinician and business owner.
- **Body:** Inter — clear, practical, and highly legible for experience, project evidence, and calls to action.

## Palette

| Role | Hex | Use |
|---|---:|---|
| Main teal | `#12A3A3` | Primary actions, highlights, and the monogram |
| Ink | `#1C2B39` | Main text |
| Cloud | `#F6F9FB` | Page background |
| Mist | `#E3E9EF` | Borders and quiet dividers |

## Style note

Use Fraunces for a calm, credible healthcare-and-business voice, with Inter for practical clarity. The mood is **quietly capable**: clean clinical restraint, measured teal accents, and evidence-led work that never competes with the content.

This kit matches the live portfolio at [minagayid.github.io](https://minagayid.github.io/): its current header uses Fraunces, its body uses Inter, and its existing palette includes the same cloud, ink, and teal values.
