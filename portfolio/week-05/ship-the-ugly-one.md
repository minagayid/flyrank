# Ship the Ugly One

## Live first version

**URL:** https://minagayid.github.io/flyrank/portfolio/week-04/empty-but-live/

This is deliberately a small, live first version: a clear name, role, and a path into the work. It uses the existing GitHub Pages stack so it is reachable without a paid platform or a backend.

## What is intentionally ugly / unfinished

- Case-study detail pages are still concise research artifacts rather than polished client stories.
- Contact collection is a mail link, not a form or CRM.
- The next iteration should add selected screenshots and stronger project navigation.

## Check

The page is public, responsive, and deployed from the same repository as the portfolio work.
